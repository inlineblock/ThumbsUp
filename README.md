ThumbsUp.js
=========

![ThumbsUp.js example image](https://github.com/inlineblock/ThumbsUp/raw/master/example.png)


## Easy way to run/test the index.html
```bash
$ cd ThumbsUp
$ python -m SimpleHTTPServer
```

```css
@font-face {

}
```
